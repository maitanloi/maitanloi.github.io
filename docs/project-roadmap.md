## Known Limitations & Constraints

### Current Limitations

| Limitation | Impact | Timeline |
|-----------|--------|----------|
| No search feature | Users must browse by category/tags | Q2 2026 |
| No comments | No community engagement | Q2 2026 (optional) |
| No analytics | Can't measure traffic | Q2 2026 |
| About page missing | Incomplete navigation | Q2 2026 |
| Single author | Limited perspective | Open-ended |
| Vietnamese only | Non-Vietnamese audience excluded | Backlog |

### Technical Constraints

| Constraint | Impact | Workaround |
|-----------|--------|-----------|
| Static site only | No database, no dynamic features | Use external services for comments, etc. |
| PaperMod theme | Limited to theme customization | Create custom layouts if needed |
| Hugo 0.157.0 | Pinned version may lag upstream | Update quarterly with testing |
| GitHub Pages | Uptime dependent on GitHub | Minimal risk, no SLA needed |

## Risk Assessment & Mitigations

### Content Risk

| Risk | Probability | Impact | Mitigation |
|------|-------------|--------|-----------|
| Low writing consistency | Medium | High | Create publishing schedule |
| Outdated content (tech changes) | Medium | Medium | Quarterly content review |
| Inaccurate HTB solutions | Low | High | Double-check exploits before posting |

**Mitigation**: Establish content calendar, peer review process.

### Technical Risk

| Risk | Probability | Impact | Mitigation |
|------|-------------|--------|-----------|
| GitHub Pages outage | Very Low | High | Have offline backup (can re-push) |
| PaperMod theme breaks | Low | Medium | Monitor releases, test updates |
| Hugo version incompatibility | Low | Medium | Lock to 0.157.0, test updates |

**Mitigation**: Monitor GitHub status, automate updates with testing.

### SEO Risk

| Risk | Probability | Impact | Mitigation |
|------|-------------|--------|-----------|
| Poor Google indexing | Medium | Medium | Submit sitemap, structure data |
| Duplicate content | Low | Low | Unique writeups, original analysis |
| Search visibility lag | Medium | Low | Wait 3-6 months for indexing |

**Mitigation**: Use Google Search Console, canonical URLs, structured data.
